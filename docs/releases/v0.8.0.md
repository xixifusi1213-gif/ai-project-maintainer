# v0.8.0: Standards crosswalk and trust evidence

This release adds standards-backed trust metadata to reports.

## Highlights

- Adds `TRUST.md`, `DESIGN.md`, and `docs/STANDARDS-CROSSWALK.md`.
- Adds report `standards.sources` and `standards.mappings`.
- Adds per-check `standardRefs` and `evidenceLevel`.
- Adds Markdown report sections for `Evidence Levels` and `Standards Crosswalk`.
- Keeps SARIF focused on actionable findings and avoids uploading standards text as code-scanning noise.

## What This Is Not

This is not a compliance certification, security guarantee, penetration test, or replacement for production ownership.

The standards mapping explains why checks exist and which public engineering practices they support.
