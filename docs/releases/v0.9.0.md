# v0.9.0: AI agent and MCP risk checks

This release adds a local-only preflight for AI-assisted maintenance workflows.

## Highlights

- New `agent-risk` command:
  ```powershell
  npx ai-project-maintainer agent-risk "E:\my-project"
  ```
- New gate integration:
  ```powershell
  npx ai-project-maintainer gate "E:\my-project" --agent-risk --strict --release
  ```
- Scans MCP, Codex, Claude, Cursor, and agent instruction/config surfaces.
- Detects inline token exposure, broad MCP permissions, unpinned MCP server packages, unsafe agent instructions, prompt-injection-like repository text, and dangerous project scripts.
- Adds `agentRisk` to JSON/Markdown reports.
- Maps agent-risk findings to OWASP LLM Top 10, OWASP Agentic AI, NIST SSDF, and OWASP SAMM.

## Safety Boundary

`agent-risk` does not call OpenAI/Codex APIs, does not start MCP servers, and does not execute project scripts. It only reads local repository files and redacts sensitive values from reports.
