# v1.3.0: Real case library and public benchmark

v1.3.0 expands the real OSS case studies into a reproducible public benchmark.

## Added

- `npm run benchmark:verify`
- `examples/benchmark-cases/`
- `docs/BENCHMARK.md`
- `docs/BENCHMARK.zh-CN.md`
- committed benchmark snapshot under `docs/benchmark-output/`

## Benchmark Coverage

- Electron desktop: SiYuan Electron RCE
- Database: Ghost Content API SQL injection
- Web/API: Next.js middleware authorization bypass
- CI / supply chain: tj-actions/changed-files compromise
- OSS npm library: TanStack npm package compromise postmortem

## Changed

- `cases:verify` remains compatible through the benchmark runner's real OSS subset.
- Benchmark cases now generate before reports, repair-pack output, after reports, and per-case summaries.

## Compatibility

- No CLI breaking changes.
- No new external accounts, scanners, or AI APIs.
- Benchmark output is evidence-oriented, not an exploit corpus or production safety guarantee.
