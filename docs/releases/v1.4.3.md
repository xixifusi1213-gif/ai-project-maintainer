# v1.4.3: Quickstart Semgrep calibration

v1.4.3 is a small first-run usability patch based on real public project smoke tests after v1.4.2.

## Changes

- Keep Semgrep enabled in quickstart.
- Treat selected Semgrep supply-chain hardening findings as non-blocking quickstart warnings:
  - GitHub Actions mutable action tags.
  - Dependabot missing cooldown.
  - npm missing minimum release age.
- Keep other Semgrep findings blocking in quickstart.
- Keep strict release gates conservative: Semgrep failures still block outside first-run quickstart.
- Show warning details in `quickstart-summary.md`, so downgraded hardening findings are still visible.

## Boundary

This release does not disable Semgrep and does not relax strict release gates. It only reduces first-run quickstart friction for hardening recommendations that are better handled as release-readiness work after the user gets an initial report.
