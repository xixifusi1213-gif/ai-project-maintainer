# v1.2.1: Repair loop dogfood

v1.2.1 proves the AI Agent Repair Pack loop end to end on the bundled demo project.

## Added

- New deterministic repair-loop demo:

  ```powershell
  npm run smoke:repair-loop
  ```

- The demo runs:
  - broken temp demo copy
  - production gate
  - repair-pack generation
  - simulated AI repair
  - `npm test` recheck
  - final production gate

- CI now runs the repair-loop smoke test.

## Changed

- README and demo docs now show the full `gate -> repair-pack -> simulated AI repair -> recheck -> gate` loop.
- `docs/REPAIR-PACK.md` explains which file to give Codex, Cursor, Claude Code, and Cline.
- Codex skill Quick Start numbering is fixed.

## Compatibility

- No CLI breaking changes.
- No new external accounts, AI APIs, scanners, or production connectors.
- Final demo status remains `PASS_WITH_GAPS` because production evidence gaps stay visible after deterministic blockers are fixed.
