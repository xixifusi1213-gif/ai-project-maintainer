AI Project Maintainer

90-second release readiness demo for AI-coded projects

1. Start with project profile

npx ai-project-maintainer init-audit .\examples\demo-ai-app
project facts -> business flows -> evidence sources -> risk policy

2. Before state: behavior is broken

node .\examples\demo-ai-app\scripts\create-before-state.mjs
FAIL expensive orders require manual review
FAIL orders cannot release without stock

3. Fix deterministic blockers

npm test --prefix .\examples\demo-ai-app
PASS checkout quote preserves customer-visible total
PASS order release respects payment, stock, and risk

4. Run production gate

npx ai-project-maintainer gate . --production --strict --release
PASS tests, build, secrets, dependencies, SAST, CI
GAP approval, monitoring, logs, metrics, alerts

5. Produce evidence report

reports/security-report.json
reports/security-report.md
reports/security-report.sarif
reports/sbom.cdx.json

6. Dogfood in GitHub Actions

CI: green
Security: green
